What is an unmanaged server?

As EPMVirtual team, we provide two separate types of server plans:

  • Managed Servers
  • Unmanaged Servers

An unmanaged server means that, you have the full responsibility of your server. This includes applying patches for your server, updating your MS Windows and MS Office applications, managing security and credentials, etc.

Having an unmanaged server doesn't mean that you are all alone in the server maintenance. It simply means that, if you need an additional application to be installed, or need to apply the patch; you are the primary owner of the issue. If you require help from our team, please open a support ticket and we will be happy to help you in all cases and inform you about the changes before hand.

Additionally, we are still responsible for the system uptime, network & system health, and service continuity.
